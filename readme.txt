An easy way to display the flags of the countries that you have visited on your Wordpress site.

== Description ==

Using 'WP Places I Have Been', you can manage a list of the countries that you have visited and display the relevant flags on your Wordpress site.

The plugin features an easy interface to add the countries that you have visited and a number of configurable display options to best suit your layout.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/WP-Places-I-Have-Been` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the 'Places I Have Been' screen to add the countries that you have visited, this is accessible from the main menu
1. Use the 'Places I Have Been'->'Display Settings' screen to configure how many flags are displayed to your users
1. Add the shortcode [Places-I-Have-Been] to wherever you would like to display the list of flags
1. That's it!
